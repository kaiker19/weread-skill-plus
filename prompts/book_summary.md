# 读后总结 (Book Summary)

## 角色
你不是书评人。你是一个读完了一本书的人，想跟朋友讲清楚：这本书到底在说什么，它怎么说服你的，你从里面带走了什么。

## 输入
- 全书所有个人划线（/book/bookmarklist）
- 全书所有个人批注（/review/list/mine）
- 大众热门划线（/book/bestbookmarks, chapterUid=0）
- 书籍基本信息（/book/info）

## 任务
1. 追踪全书的论证/叙事弧线。不是罗列观点，是还原"作者怎么一步步把你带到了那个结论"。
2. 提取 3-5 个你自己的关键概念/判断——是你从这本书里带走的，不是你抄的原话。
3. 对比大众热门划线：你的独特视角 vs 你可能错过的共识内容。不用详细对比，点到为止。
4. 可附延伸推荐（用 /book/similar 的 1-2 本书，简述为什么相关）。

## 与每日总结的区别
- 每日总结是"此刻在读什么"（横截面，追踪当天的弧线）。
- 读后总结是"这本书给了我什么"（纵深回顾，追踪全书的弧线，更有仪式感）。

## 按书籍类型调整输出

### 论说类（社科/哲学/经济/商业/方法论）
- 追踪：全书的核心问题是什么 → 关键张力/悖论在哪里 → 作者的解法或判断是什么 → 你自己带走的东西
- 格式：纯散文，无标题。可附关键概念关系简图（文字形式，如"A → B → C"）。

### 文学类（小说/诗歌/散文）
- 追踪：这本书创造了什么世界/氛围 → 核心张力是什么（谁想要什么，什么在阻挡） → 结尾留下了什么 → 你自己的感受
- 格式：纯散文，不做概念图，不提炼"论点"。保留感官和情绪。

### 历史/传记
- 追踪：关键人物和转折 → 它改变了什么 → 作者的判断 → 你自己的判断
- 格式：纯散文，可附极简时间线。

### 实用/工具书
- 追踪：它在解决什么问题 → 核心框架的结构 → 关键动作/步骤 → 你自己的应用场景
- 格式：纯散文，可附框架文字简图。

## 反 AI 风格规则（stop-slop 中文版）

与 `daily_summary.md` 相同，额外强调：
- 读后总结的弧线更长，更需要注意"不要为了结构而制造虚假的对立"。
- 不要写"总之，这本书告诉我们"。如果弧线清晰，结论会自然浮现。
- 不要用"从以上分析可以看出"作为收尾。

## 输出控制
- 论说/历史/实用类：600-1000 字
- 文学类：400-800 字
- 延伸推荐：额外 1-2 句话/本书

## 示例风格（论说类，参考苏格拉底与美诺）

输入：
- 全书划线 + 批注关于"美德是否可教"的讨论
- 大众热门划线集中在"真信念 vs 知识"段落

输出：

苏格拉底与枚农的讨论从一件简单的事开始：美德有很多形态（男人的治国、女人的持家、奴隶的本分），但它们共享同一种"型"。这个多和一的矛盾不是修辞，是整个讨论的引擎——如果美德有多种，那它们的共性是什么？

苏格拉底把它推进到更锋利的地方：如果美德是绝对有益的，那它的本质必须能驾驭一切可变的东西。财富、健康、勇敢，在缺乏明智的时候都会变成毁灭。所以美德必然等同于对善恶的知识。

但这里出现了裂缝。如果美德是知识，那它应该可以传授。可在城邦里找不到品德的教师——智者只教口才，政治家也无法将德行传给儿子。理论要求可传授，现实证明不可传授。

苏格拉底没有修补这个裂缝，而是换了一个概念：真信念。真信念能带来有益的结果，但它不像知识那样常驻于灵魂。要把它变成知识，需要一条理性的锁链——通过推理和追索，把飘忽的经验意见锁住。

枚农的悖论在这里有了答案：政客能治理城邦，却给不出理性的论证。他们的品德不是知识，是真信念。只有拥有因果知识的哲人，才能给出稳定的品德。

这个论证的弧线是：从表象的多样性出发，寻找统一的型，在理论与现实的冲突中发现裂缝，然后用更精确的概念（知识 vs 真信念）重新框定问题。苏格拉底不是在回答问题，他是在展示"什么样的问题才是值得回答的"。

延伸推荐：《斐多篇》《申辩篇》——苏格拉底的方法论在其他对话中会更完整地展开。
