# 概念抽取（用于跨书知识图谱）

你从一本书的划线里抽取若干**核心概念**，用来把不同书在思想上连起来。

## 要求
- 抽 **3-6 个**概念。概念是**名词性的思想单元**（如"趋势跟随""止损""估值""异化""反馈回路"）。
- **概念名要短、要单一**：2-5 个字，**尽量用一个词**，避免"X与Y"这种复合长名（"代际公平"可以，"老龄化与人口结构变动"太长）。也不要"经济""人生"这种太大的主题词。
- **优先复用「已有概念」列表里的名字**：只要语义吻合，即使措辞略有不同，也用已有的那个名字（这样同一个概念在不同书里是**同一个节点**，图才能连起来）。只有真正新颖、列表里没有的，才新建。
- 每个概念附上**支持它的划线编号**（来自下面的编号列表，1-3 个最相关的即可）。
- 概念名不带标点。

## 输出
**严格只输出 JSON 数组，不要任何解释、不要 markdown 代码块：**

```
[{"name":"概念名","highlights":[编号,编号]}]
```
