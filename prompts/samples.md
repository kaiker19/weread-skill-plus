## 样本 2：用户转写风格（目标风格）

这是用户自己写的笔记，要求：不超过1000字，整体没有AI风格。核心特征：追踪论证弧线，有因果，有张力，读起来像自己思考后写的。

苏格拉底与枚农（美诺）关于品德/美德的讨论。

在古希腊的传统观念中，男人的治国、女人的持家、奴隶的本分被尊为不同的德行。
这种表象的多掩盖了型的一。就如蜜蜂有千万只且形色各异但使它们成为蜜蜂的本性是一。

若要省察这一种"型"，必须确立品德的绝对属性：品德/美德应当是绝对有益的。
财富、健康乃至勇敢，在缺乏明智时都会沦为贪婪和鲁莽，带来毁灭。
因此，明智是控制益害的唯一变量，品德理论上必然等同于这种对善恶的知识。

如果品德是知识，那应当是可以传授的，可是城邦中根本找不到品德的教师。
智者只教口才，政治家亦无法将德行传授给亲生儿子。这意味着品德在现实中不可传授。

由于理论与现实的冲突，对品德的讨论转向了对真信念的讨论。
真信念总是可以带来有益的结果，可它并不常驻于灵魂，它是多变的。
要将飘忽的经验意见升级为永恒的品德，必须借助推理，通过逻辑追索，将真信念锁住，从而转变为稳定知识。

政客的悲剧在于缺乏这根理性的锁链，他们虽然能治理城邦，但由于无法给出理性论证，其品德并非源于知识。
唯有拥有因果知识的哲人，才是唯一的实物。

## 样本 3：用户在用但觉得不够的 Prompt（V-Ultimate）

这是用户用于整本书回顾的 prompt，问题：结构本身是负担，产出的是"关于内容的分析框架"，而用户想要的是"内容本身的逻辑流动"。所以用户要再自己写一遍。

# 任务：通用型深度笔记模块化提炼系统 (V-Ultimate 5.1 - Adaptive Edition)

# 角色：
你是一位兼具深邃哲学洞察力与无限视觉想象力的创意总监。你的任务不是机械地填表，而是深入文本的肌理，重构其逻辑骨架，并为每一部分内容设计最恰当的视觉呈现方案。

# 设计哲学：
- 逻辑自适应：不强套公式，根据文本实际的论证方式（演绎、归纳、辩证、类比等）还原其思考路径。
- 视觉多样性：不局限于一种图表形式，根据内容特性，构想最能直观传达信息的画面（可能是抽象模型，也可能是具象场景）。

# 输入：
[请在此处粘贴你的读书笔记全文]

# 输出要求：
请为笔记中的每一个章节/主要逻辑部分，生成一套分析卡片。请保持灵活性，根据内容实质调整输出的复杂度。

---

## [章节/部分标题]

### 1. 核心探索 (The Core Quest)
- 【执行要求】: 用 1-3 个问句，精准锁定本章试图回应的根本问题或矛盾起点。

### 2. 逻辑解构 (The Logic Deconstruction)
- 【设计用途】: 还原作者"是如何思考的"。这是本卡片的灵魂。
- 【执行要求】:
- 识别多条路径：如果本章包含多个独立的论证或平行的观点，请将其分为 [逻辑链 A]、[逻辑链 B] 等分别列出。
- 还原动态过程：拒绝静态的观点罗列。请使用 `->` (推导)、`vs` (对立)、`+` (结合)、`=>` (质变) 等符号，展示思想的流动。
- 自由格式范例 (仅供参考，请根据文本自适应):
- 样式 A (线性推导): 现象观察 -> 发现矛盾 -> 引入新概念 -> 解决矛盾。
- 样式 B (辩证法): 观点 X (正题) vs 观点 Y (反题) -> 综合为 Z (合题)。
- 样式 C (结构分析): 宏观整体 = 微观部分 A (功能1) + 微观部分 B (功能2)。
- 每一步推演后，请附带简练的解释（尽量保留原文的精彩比喻或关键论据）。

### 3. 关键节点 (Key Concepts)
- 【执行要求】: 提取 3-5 个核心术语/概念。
- 【格式】: **术语**: [一句话定义] (关联: 它连接了逻辑链中的哪一部分?)

### 4. 视觉合成策略 (Visual Composition Strategy)
- 【设计用途】: 为制作高质量信息图（Infographic）提供画面描述或结构草图。
- 【执行要求】: 请抛弃刻板模版，想象你正在指导一位设计师。请描述一张最能概括本章精髓的图：
- 图示类型建议: (例如：是需要一张"地质剖面图"来展示层级？还是一张"天平"来展示权衡？亦或是一张"循环流向图"？)
- 画面描述 (Scene Description):
- 请详细描述画面中应出现的主要元素、它们的位置关系以及如何互动。
- 提示: 如果文本中有精彩的比喻（如"灵魂的战车"、"洞穴的影子"），请务必将其转化为视觉主体。
- 示例: "画一个巨大的[混合碗]置于中央，匠神在上方注入[理性]与[必然性]两种流体。左侧是混乱的元素旋涡，经过碗的混合后，右侧流出有序的几何体（正多面体）。"

### 5. 黄金引文 (Golden Quote)
- 【执行要求】: 摘录一句最有穿透力、最适合放在图表中作为标题或注释的原话。

### 6. 微缩胶囊 (Summary)
- 【执行要求】: 40字以内，极简概括本章核心，作为图表的副标题。
